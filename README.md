# SQL-Consumer_Goods_Analysis

# Company Overview
**AtliQ Hardwares** is a fictional yet industry-leading computer hardware manufacturer headquartered in India with operations worldwide. The company embraces data analytics to make smarter, data-driven business decisions and enhance operational efficiency.

# Problem Statement
AtliQ Hardwares recognized delays in making strategic, data-backed decisions across departments. To strengthen their analytics function, the company initiated a hiring program for Junior data analysts with solid technical and problem-solving abilities. To assess candidates fairly, Tony Sharma, Senior Data Analyst, created a **SQL challenge** designed to test real-world analytical skills

# Project Objective
* Write SQL queries to answer 10 **ad-hoc** business questions.
* Analyze sales, inventory, and performance data.
* Provide insights to support better decision-making.

# Results
This project demonstrates my ability to work with complex data sets, write optimized SQL queries, and present insights clearly. It not only strengthened my technical skills but also improved my ability to communicate data-driven findings effectively.

# Repo Structure
* [Ad-hoc Requests](ad-hoc-requests.pdf): This document lists the ten ad-hoc business requests for this challenge.
* [Metadata](Metadata.txt): This provides an overview of the tables used in this analysis
* [Hints](Hints.pdf): This file contains hints to refer to when someone gets stuck writing the queries.
* [Queries](Query_Requests.sql): It contains the complete list of all 10 ad-hoc requests.
* [Presentation](SQL_Project_RC-4.pptx): This presentation includes query results and insights for decision-making.
